# User Guide

## Starting a Scan

1. Login to Hawkeye.
2. Navigate to the Dashboard.
3. Enter the website URL.
4. Click **Scan**.
5. Wait for the analysis to complete.

---

## Viewing Reports

- Open Scan History.
- Select a completed scan.
- Review the findings.
- Download the PDF report if required.

---

## Security Score

The overall security score is calculated based on:

- HTTPS Configuration
- SSL Certificate
- Security Headers
- Cookie Security
- Website Configuration
